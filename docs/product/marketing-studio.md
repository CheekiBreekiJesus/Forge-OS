# Marketing Studio Product Notes

Date: 2026-07-01

## Demo Scope

Marketing Studio gives a manufacturing SME a single local workspace to prepare marketing assets and campaigns before any external publishing.

Available sections:

- Overview
- Campaigns
- Image Studio
- Asset Library
- Brand Kit
- Audiences
- Advertising Accounts
- Analytics
- Video Studio

## User Workflow

1. Create or refresh the Brand Kit from the local company profile.
2. Create an audience.
3. Create a campaign linked to products and channels.
4. Generate deterministic campaign copy.
5. Upload or generate local preview image assets.
6. Approve reviewed assets and campaigns.
7. Export JSON or CSV review packages.
8. Review local analytics and provider diagnostic status.
9. Create a storyboard-only video project.

## What Is Intentionally Disabled

- Live ad publishing.
- Live provider budget spend.
- Paid image generation.
- Paid video generation.
- External metric sync.
- Automated posting to social networks.

## Localization

The UI supports `pt-PT` and `en`. Spanish is not translated yet, but the dictionary-backed structure is compatible with adding `es` later.

## Demo Data

The current implementation uses local synthetic/demo records only. Real customer lists, supplier files, private pricing, and confidential documents must remain outside Git.
